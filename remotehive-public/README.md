# Remotehive-New

A monorepo for RemoteHive using a remote-first architecture. This repository contains:
- `remotehive-admin`: Admin panel built with Next.js + TypeScript + Tailwind.
- `remotehive-public`: Public website built with React (Vite) + TypeScript + Tailwind.
- Supabase used for database, auth, and storage.

## Steps Completed
- Step 1: Repository initialization and architecture selection.
- Step 2: Frontend scaffolds created for Admin and Public sites.
- Step 3: Shared logo asset paths established for both frontends.
- Step 4: Supabase client scaffolds and `.env.example` added.
- Step 6: Supabase Schema and Policies setup.
- Step 7: Profile Page and AI features implemented.
- Step 8: Appwrite Integration for Storage and Resume Templates.
- Step 9: Advanced Resume Builder (10 Templates, Cloud Save, Image Download).

See `docs/` for detailed step-by-step documentation.
